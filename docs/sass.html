Units

We use the extend syntax in SASS to clear floats. Simply apply @extend %cf; to the element that the floats are within.

The example would contain all floats within .wrapper.

.wrapper {
	@extend %cf;
	.content,
	.sidebar {
		float: left;
	}
}

Media Queries

Media Queries in Air-Drop should be used with the module that they modify. There is a mixin provided that makes it much easier to keep the styles together (and deliver the appropriate files to IE8 and below).

The mixin can be used in two ways: with a single element and surrounding multiple elements. Both variations can be seen in the example.

The are 5 parameters that can be passed to the mixin to modify its behavior – only one of which is required. They are as follows:

  1. Point(s): one breakpoint is required, but multiple may be passed in a list. If more than one breakpoint is passed, the same number of parameters 4 and 5 must also be passed. Each breakpoint should be a number with a unit.
  2. For IE8: boolean that determins if the mixin will attempt to create styles for IE8 and below. Defaults to true.
  3. Stand Alone: boolean that tells the mixin if it is being used with a single element or surrounding multiple elements. Defaults to false (we assume that it is in use with a single element – see the first example).
  4. Direction(s): can be min or max. Set a direction to correlate to each breakpoint passed.
  5. Dimension(s): can be width or height. Set a dimension to correlate to each breakpoint passed.

In Air-Drop, the main screen.scss is set to not have any IE8 or below styles. A separate lte-ie8.scss file is provided to do hold styles that are used instead of media queries (and to use px instead of rem). To achieve this, the lte-ie8.scss file sets a variable ($lte-ie8: 'only';). Options for this variable are 'false' (the default, which screen.scss uses), 'true' (which would put media queries and ie8 class-based styles in the same stylesheet), and 'only' (which will only output the ie8 styles). For smaller projects, you may want to set this to 'true' for the main stylesheet (I recommend overwriting the variable in your _settings.scss file). 

a {
	@include mq(500) {
    	color: $black;
	}
}

@include mq((500, 800), true, true, (min, max), (width, width)) {
	a { color: $black; }
	em { color: $black; }
	strong { color: $black; }
}
